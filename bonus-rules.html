всё о бонусах

Как накопить бонусы?
Накопление бонусов происходит за онлайн покупки на сайте palladium.ua. За каждую покупку* онлайн на бонусный счет клиента зачисляется 1% от суммы.
Магазином регулярно проводятся акции, розыгрыши, где в качестве приза начисляются бонусы на счета клиентов. Накопленные бонусы можно использовать как оплату за свой заказ. Бонусная программа позволяет участникам экономить на покупках, участвовать в приобретении специальных акционных товаров.
*Исключения, когда бонус 1% не начисляется:
- при заказе сервиса (например, по подключению техники и др.);
- при покупке подарочного сертификата;
- при погашении части стоимости товара с помощью сертификата;
- в других предварительно оговоренных случаях.
Как потратить бонусы?
Потратить бонусы можно на новые онлайн покупки в интернет-магазине palladium.ua.
Зачисление бонусов в счет покупки возможно до 100% стоимости нового товара.
Использовать свои бонусы можно через 14 дней после их начисления. Количество бонусов, которыми можно воспользоваться показывается в поле бонусов “Доступно”.
Бонусы действуют до 31.12 текущего года, по истечении срока они аннулируются.
Как узнать остаток бонусов?
Количество бонусов отображается в личном кабинете. Если бонусы уже можно использовать, они попадают в пункт «Доступно». В пункте «Всего» отображается общее количество бонусов, которое будет доступно по истечении установленного срока.
Когда можно использовать бонусы?
Бонусы можно использовать через 14 дней после покупки и зачисления их в пункт «Доступно». Если прошло более 14 дней, а бонусы не зачислились в пункт «Доступно», просим связаться с нашим магазином по телефону: 0800 211-233 (номер бесплатный).
Доступны ли бонусы для офлайн покупок (если заказ происходит без оформления через корзину)?
На данном этапе зачисление/погашение бонусов возможно только при оформлении заказа онлайн на сайте через корзину.
Какой срок действия бонусов?
Срок действия бонусов до 31.12 текущего года. По истечении указанного периода (с 01.01 следующего года) бонусы аннулируются.
При возврате товара, что происходит с бонусами?
Если произошел возврат товара, то бонусы, начисленные за него, снимаются со счета.
